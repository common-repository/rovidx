== Description ==
The **RoVidX Media Framework** allows Wordpress installations to easily distribute video content.  If you have ever wanted to open your own video portal, but found the Wordpress offerings lacking for video this is the plugin for you.

This plugin is the base for the framework.   It will allow you to input information that our output modules will require.

For more information [check out our Website](http://rovidx.com)!

== Installation ==
1. Upload & Activate Plugin
2. Add content (RoVidX content options are located in the post editor)
3. Choose add-ons to output your content

[Download Add-Ons Here](http://rovidx.com/add-ons/ \"RoVidX Add-ons\")

== Frequently Asked Questions ==
= Does this work with audio? =

Not currently but we will support it if the demand is there.

= How do I get this feed on Roku/Boxee/Plex? =

Download the Roku VOD Lite plugin and follow the instructions located in the administration dashboard menu.

= What\'s the difference between LITE and PRO versions of Addons? =

The lite versions of the plugins are aimed towards the small independent video creator who is a hobbyist.

The Pro versions are aimed towards the small to medium sized video production companies.

== Screenshots ==
